# ObraPro Enterprise

Sistema completo de gestión de obras para construcción.

## Características

- ✅ **Múltiples obras/proyectos**
- ✅ **Mediciones** con calculadora integrada
- ✅ **Control de horas** con fotos y geolocalización
- ✅ **Gestión de materiales**
- ✅ **Personal y equipos**
- ✅ **Control de gastos**
- ✅ **Facturación y presupuestos**
- ✅ **Exportación PDF profesional**
- ✅ **Exportación Excel**
- ✅ **Funciona 100% offline**
- ✅ **Sincronización en la nube** (Firebase)
- ✅ **Autenticación de usuarios**
- ✅ **Backup y restauración**

## Instalación

1. Copia todos los archivos a tu repositorio de GitHub
2. Genera los iconos abriendo `icons.html` en tu navegador
3. Guarda los PNG generados en la raíz del proyecto
4. Activa GitHub Pages en Settings → Pages
5. Abre la URL en tu móvil y selecciona "Agregar a pantalla de inicio"

## Configuración Firebase (Opcional)

1. Crea proyecto en [Firebase Console](https://console.firebase.google.com)
2. Copia la configuración en `index.html`
3. Habilita Firestore Database y Storage
4. Configura reglas de seguridad

## Uso

### Primera vez
1. Crea tu cuenta de usuario
2. Crea tu primera obra
3. Comienza a registrar mediciones, horas, etc.

### En obra
- Usa el **teclado calculadora** para sumar mediciones rápidamente
- **Toma fotos** al registrar horas como evidencia
- Consulta el **dashboard** para ver progreso en tiempo real

### Exportar datos
- **PDF**: Informe profesional completo
- **Excel**: Para contabilidad y análisis
- **WhatsApp**: Resumen rápido para clientes

## Tecnologías

- React 18
- Tailwind CSS
- IndexedDB (almacenamiento local)
- Firebase (sync opcional)
- jsPDF (generación PDF)
- SheetJS (exportación Excel)
- Chart.js (gráficos)
- Service Workers (PWA)
